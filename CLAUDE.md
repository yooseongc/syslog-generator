
# 프로젝트 개요

 - linux 상에서 syslog 부하를 만드는 프로그램을 개발합니다.
 - syslog 전송 속도 (초 당 N 개), 로그의 크기 (K bytes)에 따른 부하를 재현하고 그에 따른 시스템 부하를 측정하는 것이 목적입니다.
 - linux 상에서 CLI로 동작합니다. 가능하면 설정 변경이나 모니터링까지 같이 하는 TUI이면 좋습니다.

# 제약 조건

 - musl libc로 빌드되는 Rust 1.93 
 - syslog syscall을 사용합니다.

# 개발 시 준수 사항

 - PROCESS.md에 작업 시작 시, PLAN 수립 완료 시, 개발 시작 시, 개발 종료 시 작업 내용에 대해서 요약 정리합니다.
 - NOTES.md에 작업시 주의 사항, 개발자와의 협의 사항 등을 정리합니다.
 - 매 작업 시 반드시 PROCESS.md, NOTES.md 를 읽고 시작합니다.
 - 사용법 및 시험 방법을 CLI.md에 정리합니다.
